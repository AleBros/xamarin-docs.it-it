# <a name="message-app-integration-in-xamarinios"></a>Integrazione di App di messaggi in xamarin. IOS

_Mostra questi articoli includono come un'estensione dell'App messaggi in una soluzione xamarin. IOS che si integra con l'app Messages per presentare le nuove funzionalità all'utente._

Nuovo ai dispositivi iOS 10, un'estensione dell'App messaggi si integra con il **messaggi** app e presenta nuove funzionalità per l'utente. L'estensione può inviare testo, le etichette, i file multimediali e i messaggi interattivi.
