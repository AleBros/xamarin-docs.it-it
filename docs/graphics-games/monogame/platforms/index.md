# <a name="monogame-platform-specific-considerations"></a>Considerazioni specifiche della piattaforma monogame

## <a name="monogame-on-universal-windows-platform-uwpgraphics-gamesmonogameplatformsuwpmd"></a>[MonoGame su piattaforma UWP (Universal Windows Platform) (UWP)](~/graphics-games/monogame/platforms/uwp.md)

Questa procedura dettagliata illustra la creazione di un progetto MonoGame piattaforma UWP (Universal Windows Platform) (UWP) e il caricamento del contenuto. Le app UWP possono essere eseguite su tutti i dispositivi Windows 10, tra cui desktop, tablet, Windows Phone e Xbox One.
