# <a name="introduction-to-macos-high-sierra"></a>Introduzione a macOS High Sierra

macOS High Sierra introdotte nuove funzionalità e aggiornamenti per macOS, tra cui:

- HEVC (codifica Video ad alta efficienza)
- Apple File System
- Bare metal 2
