# <a name="introduction-to-macos-high-sierra"></a>Introduzione a macOS Sierra elevata

macOS High Sierra introdotte nuove funzionalità e gli aggiornamenti a macOS, tra cui:

- HEVC (ad elevata efficienza la codifica Video)
- Apple File System
- Metal 2
