Note sull'esecuzione di test
Xamarin.iOS
  • Approccio più semplice per iniziare a usare i simulatori per iPhone, iPad, Apple Watch e Apple TV inclusi con Xcode. I simulatori sono accessibili nel Mac connesso durante il debug con Visual Studio.
  • Per usare un dispositivo per il test, seguire queste istruzioni (eseguendo la maggior parte dei passaggi nel computer Mac connesso).
Xamarin.Android Per configurare il dispositivo o usare un emulatore, seguire queste istruzioni:
  • Il programma di installazione di Xamarin include Gestione emulatori Googleche permette di configurare emulatori Android di Google per l'esecuzione di test.
  • Altri emulatori sono disponibili da Genymotion e Microsoft (con Visual Studio 2015).
Xamarin.Forms Le app Xamarin.Forms possono essere distribuite nei dispositivi e negli emulatori pertinenti come descritto sopra. L'app iOS può essere testata solo tramite l'hardware Mac connesso, mentre le app per tablet/desktop Windows per Windows 8.1 o UWP possono essere testate direttamente nel computer di sviluppo.
